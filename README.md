# Usage

```python
from timise import TIMISE

timise = TIMISE()
timise.evaluate("/home/user/model_xx_prediction","/home/user/gt", "/home/user/output", data_resolution=[30,8,8])
timise.summarize()  # Summarize the results as tables in the console
```
